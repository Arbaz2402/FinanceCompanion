# FinanceCompanion 💰

> **A premium, privacy-focused financial companion for the modern iOS experience.**

Built with **SwiftUI** and **SwiftData**, FinanceCompanion combines professional-grade financial tracking with a beautiful, intuitive interface.

---

## 🌟 App Highlights

- 🔒 **Privacy-First**: No accounts, no cloud, no trackers. Your data never leaves your device.
- 📈 **Smart Insights**: Native Swift Charts visualize your spending patterns instantly.
- 🎯 **Goal Oriented**: Gamified savings goals with streaks and visual progress tracking.
- ⚡ **Premium UX**: Fluid animations, haptic feedback, and a clean MVVM-based architecture.

## ✨ Core Features

<details open>
<summary><b>🏦 Intelligent Dashboard</b></summary>

- **Unified Snapshot**: Instantly view balance, income, and expenses in **INR (₹)**.
- **Spending Analysis**: Interactive bar charts for your top 5 categories.
- **No-Spend Streak**: Gamified discipline tracking with a visual streak counter.
- **Goal Carousel**: Snapping horizontal monitor for active savings goals.
- **Daily Tips**: Personalized insights to improve your money habits.
- **Welcome Context**: Time-aware greetings for a personal touch.
</details>

<details>
<summary><b>📊 Smart Insights & Patterns</b></summary>

- **Time-Range Toggles**: Switch between **Day**, **Week**, and **Month** trends.
- **Pattern Analysis**: Identifies top spend, trend % change, and frequent habits.
- **Category Breakdown**: Immersive sector charts for visual expense decomposition.
</details>

<details>
<summary><b>📝 Seamless Transaction Management</b></summary>

- **Immersive Entry**: Redesigned forms with focused layouts and localized inputs.
- **Advanced Filtering**: Drill down by month, date, or transaction type.
- **Global Search**: Quick keyword-based search across notes and categories.
- **Tactile Feedback**: Integrated haptics for category selection and confirmations.
</details>

<details>
<summary><b>🎯 Goal-Oriented Savings</b></summary>

- **Dream Tracking**: Create goals with target amounts and optional deadlines.
- **Auto-Bookkeeping**: Goal contributions automatically sync with your balance.
- **Savings History**: Dedicated history view for individual goal contributions.
- **Dynamic Progress**: Visual bars that adapt as you approach your target.
</details>

## 🚀 Getting Started

### 1. Prerequisites
- Xcode 15.0+
- iOS 17.0+ (Simulator or Physical Device)

### 2. Installation
```bash
# Clone the repository
git clone https://github.com/Arbaz2402/FinanceCompanion.git

# Open the project
cd FinanceCompanion
open FinanceCompanion.xcodeproj
```
*Press `Cmd + R` in Xcode to build and run.*

## 📝 Design & Architecture

<details>
<summary><b>Deep Dive into Technical Decisions</b></summary>

### 🏗 Architecture
- **Clean MVVM**: Strict separation of concerns using the modern `@Observable` framework.
- **Modular Views**: Refactored view hierarchy with extracted subviews for performance.

### 🎨 Design & UX
- **Color Palette**: Professional mix of system colors and gradients for a premium feel.
- **Localized**: Native support for **Indian Rupee (₹)** formatting.
- **Automatic Savings**: Integrated goal-transaction syncing to reduce manual friction.
- **Persistence**: Schema-driven local storage for reliable data management.
</details>
